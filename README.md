# Febuxostat-S1PR1-In-Silico
An integrated computational pipeline combining Machine Learning (Random Forest), Structure-Based Virtual Screening, and Molecular Dynamics (GROMACS) to identify S1PR1 modulators. Features the discovery of Febuxostat as a high-affinity repurposing candidate for Multiple Sclerosis therapy via the 7EW0 active-state signaling complex.
